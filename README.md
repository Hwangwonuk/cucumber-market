# cucumber-market
## 개발목표

웹 UI는 카카오 오븐으로 대체하여 프론트엔드 부분은 생략하고 벡엔드에 초점을 맞춰 백엔드 개발에 주력   
단순 기능 구현 뿐 아니라 코드의 재사용성 및 유지보수성 등을 고려하여 구현하는 것을 목표로 개발   
ex) 당근마켓 같은 중고거래 사이트
